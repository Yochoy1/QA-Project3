# 找回密码功能测试点

1.  **正常流程测试**：输入已注册的手机号/邮箱，点击「发送验证码」，能成功收到验证码；输入正确验证码和新密码，能成功重置密码。
2.  **异常场景测试**：输入未注册的手机号/邮箱，系统应提示「该账号未注册」，不发送验证码。
3.  **边界值测试**：输入格式错误的手机号（如少一位、含特殊字符），系统应实时提示「手机号格式不正确」。
4.  **安全性测试**：验证码有效期内可使用，超时后验证码失效；同一账号短时间内多次请求验证码，应触发频率限制。
5.  **密码规则测试**：新密码不符合复杂度要求（如长度不足、无大小写字母）时，系统应给出明确提示，无法提交。
